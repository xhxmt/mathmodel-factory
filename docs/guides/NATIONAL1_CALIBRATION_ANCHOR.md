## 锚点 1：2024A 国一摘要全文（A 题写作范本）

> 板凳龙作为浙闽地区的一种传统民俗活动，以其高观赏性和舞龙队的协调配合著称。本文针对板凳龙表演的三个阶段盘入、调头和盘出，通过分析舞龙队的行进路线及各节板凳之间的几何关系，建立了运动模型和行进路线优化模型。采用多种优化方法，对舞龙队在不同条件下的运动状态以及行进路线进行研究。
>
> 针对问题一：构建盘入运动模型。首先，根据盘入路线为等距螺线的特征，建立极坐标系。针对龙头位置，建立有关极角的微分方程，利用微积分及二分法求解龙头前把手的极坐标，继而利用余弦定理，递推求解相邻板凳的位置关系，获得各把手的位置函数。通过对位置函数求导，得到相应的速度函数。最终，将极坐标转换为直角坐标，求得各把手在各时刻的位置和速度，结果详见表2，表3和result1.xlsx。
>
> 针对问题二：构建碰撞检测模型。首先，证明最易发生碰撞的区间为最内圈与其外侧一圈的板凳，缩小检测范围。随后，确定碰撞条件为：内圈板凳顶点与外圈板凳把手连线的距离小于板凳宽度的一半（0.15m），并通过几何关系算出距离函数，判断该函数的最小值是否小于0.15m以确定碰撞是否发生。最后，采用动态搜索策略，先用变步长搜索确定时间范围，再进行二分搜索，确定首次碰撞的准确时间。我们得到停止盘入时间为412.473838s，此时各把手的位置和速度详见表4和result2.xlsx。
>
> 针对问题三：构建最短螺距的单目标优化模型。其中目标函数和决策变量为螺距大小。在约束条件方面，基于板凳的几何参数和问题二中的参数限定螺距范围，基于盘入运动模型确定龙头前把手进入调头空间的时间，并利用碰撞检测模型判断进入调头空间前是否发生碰撞，建立相应约束。在优化求解过程中，对比后选用粒子群算法求解板间最小距离，并结合变步长搜索求得最短螺距为0.450337m。
>
> 针对问题四：基于问题一的模型，构建调头运动模型。首先证明在给定条件下，调头曲线长度为常数，无法进一步减小。随后，将调头路线划分为四个部分，通过几何关系求解出各部分的曲线方程。沿用问题一的推导方法，首先求解龙头前把手位置函数，再通过递推关系和求导计算其余把手的位置函数和速度函数。我们对调头空间附近的递推进行了详细的分类讨论，并给出了相应的递推方程。最终，将极坐标转换为直角坐标，求得各板凳把手在各时刻的位置和速度，详见表5，表6和result4.xlsx。
>
> 针对问题五：在调头运动模型的基础上，构建各把手最大速度的优化模型。首先，通过比例转换，将目标转化为在龙头速度仍为1m/s的条件下，求解各把手在所有时刻的最大速度。随后，基于问题四的速度规律及物理规律确定最大速度出现的大致时间范围，接着通过大步长搜索确定了最大速度出现的区间为[14s,15s]。由于函数在该区间内呈单峰性，应用三分搜索算法进一步精确定位最大速度。最后，通过比例关系还原龙头的最大速度，得到龙头最大速度约为1.246266m/s。
>
> 关键字：板凳龙 等距螺线 动态搜索 粒子群算法 单目标优化

### 为什么这是 90+ 分摘要（逐项解剖）

1. **每问一段，段内自成闭环**：难点/对象 → 方法链（含"为什么这样做"）→ **精确数值答案** → 交付物指针（"详见表4和result2.xlsx"）。评委 3 分钟内不翻正文就能核对每问是否答完。
2. **数值给到全精度**：412.473838s、0.450337m、1.246266m/s —— 6 位小数不是装饰，
   是"我的解可以被复现核对"的信号。A 题答案接近唯一，精度即可信度。
3. **含证明动词**："证明最易发生碰撞的区间…"、"首先证明调头曲线长度为常数，无法
   进一步减小" —— 机理论证优先于黑盒搜索，这是 A 题创新性的主要来源。
4. **方法选择有动机**："对比后选用粒子群算法"、"由于函数在该区间内呈单峰性，应用
   三分搜索" —— 每个算法出场都有一句为什么。
5. **各问段落不同构**：问题一以推导链开场，问题二以"证明+缩小范围"开场，问题四
   以"证明不变量"开场 —— 结构随每问真正的难点变化。
6. **没有泛化尾巴**：全文无"可推广到 XX 领域"式空洞结尾。

## 锚点 2：同题对照 —— 2024B 国一 vs 省一 的实证差距

两篇论文做同一道题（生产决策），奖级差一档。逐段对照后的可操作差异：

| 维度 | 国一（national_1_B） | 省一（provincial_1） |
|---|---|---|
| 摘要的答案密度 | 每问给出**完整决策方案原文**（"对零件1检测、零件2不检测、成品不检测、不合格成品拆解"）+ 利润值 + 模拟规模（1000次×2000件） | 给出结果数值，但方案表述压缩，问题四只有方法没有最终方案 |
| 方法动机链 | 二项分布→棣莫弗-拉普拉斯→方差未知→t 分布，每步转换给理由 | 直接报 Cochran 公式 + 单边检验，动机链短 |
| 正文小节组织 | 按内容分解（成本计算逐项 C1-C4） | 出现 `step1: 相关变量值的确定 … step6: 结果分析` 式**编程流水账标题** —— 工程痕迹进正文 |
| 方法层次 | 蒙特卡洛为底 + 蚁群/遗传两级优化 + 精英策略，且解释了为什么要在枚举可行时仍引入优化 | 遍历模拟为主 + 遗传算法，对比维度少 |
| 鲁棒性 | 问题四用 3σ 准则做次品率鲁棒性分析，识别敏感决策分支（零件3、4、8检测） | 敏感性分析存在但停在"证实了可靠程度"的结论句 |

**给评分的换算**：被评论文若在"答案密度 / 动机链 / 无工程痕迹"三项达到国一水平，
写作清晰度 + 结果说服力两维度方可 ≥ 17/20；只达到省一水平（结果有但方案不全、
流程式小节、验证一句带过）对应 13-15/20；更低则 < 12。

## A 题（主攻方向）特有的加分要素清单

对照 2024A 国一论文，A 题（机理/几何/连续优化类）评委期望看到：

1. **机理推导优先**：微分方程 / 几何关系 / 递推公式先行，搜索算法只做求解工具。
   纯"上来就 PSO/GA"没有推导链 = 创新性上限 ~14/20。
2. **至少一处证明或论证**：缩小搜索范围的证明、不变量证明（"调头曲线长为常数"）、
   单峰性论证。这是 A 题创新分的直接来源。
3. **全精度答案 + 逐问结果表 + result*.xlsx 指针**：三件套缺一不可。
4. **分类讨论的完整性**：边界情形（如调头空间附近的递推）逐类给方程。
5. **误差/收敛性交代**：变步长→二分/三分的精度链条要写出终止精度。
