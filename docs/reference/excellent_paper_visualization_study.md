## 材料边界

- 2025A 优秀论文：A196，`external/2025A_excellent_A2022729/`，98 页。
- 2023A 优秀论文：A0165，`external/2023A_excellent_A1865112/`，45 页。

## 总判断

优秀论文的图不是“多画图”，而是把图分配给三个高价值位置：

1. 降低模型理解门槛：几何关系、物理机理、变量定义、区域划分。
2. 支撑最终结果可信度：关键状态快照、最紧约束图、搜索/收敛/验证曲线。
3. 压缩大量数值结果：表格报告最终值，图展示趋势和空间分布。

因此，建模论文可视化的目标不是覆盖每个中间变量，而是让评委在关键节点上“看懂模型为什么这样建、结果为什么可信、最优解为什么合理”。

## 跨论文可视化共性

### 1. 先画概念或机理图，再进入公式

2025A A196 在问题一前用“有效遮挡示意图”“导弹视角下圆柱体轮廓”解释遮挡判定。2023A A0165 在定日镜模型中用太阳时角、反射向量、阴影遮挡示意图解释几何关系。2024A A242/A016/A163 也在板凳龙中用相邻把手几何关系、板凳顶点关系、调头曲线示意图解释公式来源。

规则：每个复杂几何或物理判据至少配一张“概念解释图”。这类图不需要展示所有数值，只标出对象、方向、关键点、约束边界和判定关系。

### 2. 每问通常有一张主结果图或主结果表

优秀论文常把每个问题的最终结果落在一张表或一张图上：

- 2024A A242：P1 位置/速度表 + 龙头螺线位置图；P2 终止时刻表 + 碰撞状态图；P3 `dis(t)` 函数图和粒子群收敛图；P4 调头路径和时刻状态图；P5 速度峰值曲线。
- 2025A A196：P1 离散点数误差表和有效遮蔽时长表；P2 两种算法对比表；P3/P4/P5 用投放策略表和结果表压缩高维策略。
- 2023A A0165：P1 光学效率表和空间散点图；P2 单峰曲线图和参数设计表；P3 调节后镜场排布图和参数表。

规则：每个子问题至少要有一个“最终交付视觉锚点”，可以是图，也可以是表。不要只在正文中散落数字。

### 3. 几何题偏好“路径图 + 局部放大 + 关键状态快照”

2024A 板凳龙优秀论文反复使用：

- 全局螺线/调头路径图，用颜色区分盘入、盘出、调头曲线。
- 碰撞或临界状态图，展示龙头与外圈板凳的空间关系。
- 局部放大图，说明最紧约束或碰撞点。
- 多时刻状态图，用同一坐标系展示运动过程。

规则：路径规划/碰撞/调头问题应优先画“全局路径 + 关键临界态 + 局部放大”。单纯画大量坐标表不够。

### 4. 优化题偏好“搜索过程图 + 收敛图 + 最优点标注”

2024A A242 的 P3 使用 `dis(t)` 函数图和粒子群收敛图；2023A A0165 的 P2 使用三分查找示意和单峰曲线图；2025A A196 用算法流程图和算法对比表说明优化可靠性。

规则：如果最终答案来自搜索、启发式算法或一维优化，应至少给出一张能显示“为什么搜索可信”的图：目标函数曲线、收敛曲线、可行边界曲线或算法对比表。

### 5. 空间分布问题偏好多宫格散点图

2023A A0165 的定日镜论文把多个效率指标做成 2x3 多宫格散点图，每个子图对应一个效率或输出功率，颜色条表示指标值。它的作用不是装饰，而是证明空间分布符合太阳视运动和镜场几何规律。

规则：当模型输出是“每个位置/节点/网格一个指标”时，多宫格散点图比单张表更有效。各子图必须共用坐标范围和相近配色尺度，便于比较。

### 6. 表格承担精确数值，图承担趋势和机制

优秀论文中的表格通常给出最终坐标、速度、效率、参数设计、投放策略；图则展示路径、趋势、分布、算法收敛和临界状态。二者分工清楚。

规则：不要用图替代表格中的精确数值，也不要用大表格替代趋势图。关键数字进表，趋势和机理进图。

## 对 2024A 板凳龙论文的可视化建议

### 应保留或强化的图

1. `solve_p1_inspiral_snapshots`：保留。它对应优秀论文中的全局盘入状态图，但 caption 要明确“展示 0--300s 队形沿螺线递推的合理性”。
2. `solve_p2_terminal_collision`：保留并强化。优秀论文都重视终止碰撞状态图；应标出首碰板凳对、局部放大区域和安全阈值。
3. `solve_p3_pitch_geometry` 或 `solve_p3_bisection`：二选一作为主图。主图应服务于“为什么推荐 0.450m”，不是强调粗网格下界。
4. `solve_p4_scurve_geometry`：必须保留。优秀论文 P4 都用调头曲线几何图承接公式。
5. `solve_p4_uturn_snapshots`：保留。它对应多时刻状态图，可证明整队能通过调头路径。
6. `solve_p5_speed_amplification`：必须保留。优秀论文 P5 都用速度矩阵/速度曲线定位峰值把手和峰值时刻。
7. `sensitivity_oat_p5_timestep`：保留，但标题和 caption 应写成“步长收敛性”，不要写成“风险暴露”。

### 应新增或改造的图

1. **问题关系图/子问题依赖图**：2025A A196 有“各问题关系图”。2024A 也适合用一张 DAG 表达 P1 运动学 -> P2 碰撞 -> P3 螺距 -> P4 调头 -> P5 限速。已有 `concept_subproblem_dag` 可作为这个角色，但应放在问题分析并解释每问输出如何成为下一问输入。
2. **碰撞判据几何小图**：在 SAT/AABB 细节之前，用一张矩形板凳、把手中心线、非相邻板凳、安全裕度的示意图解释实体碰撞判据。它比直接写 AABB/SAT 更接近优秀论文写法。
3. **P3 加密验证图**：把粗网格下界与加密采信值画成“采样密度 -> 最小可行螺距”的收敛/阶梯图，强调推荐值上调后稳定，而不是突出 `0.429883m`。
4. **P5 峰值局部曲线**：在速度放大矩阵之外，补一张 `t=14--15s` 附近第 3 号把手速度曲线，并标出最大倍率和反解速度。这与 A163/A242 的 P5 叙事最接近。

### 应弱化或删除的图

1. 只展示内部流程、文件或模型分支的图应弱化。优秀论文几乎不画 runner/workflow。
2. 过度暴露失败分支的对照图应后置到模型评价或局限性。主文图应服务最终采信方案。
3. 若某图只有“程序跑了”的证明，没有解释模型机制、最终结果或可信度，应删掉或移入附录。

## Step 8 后续接入建议

如果后续要把这些发现接入系统流程，建议只改 Step 8 可视化 prompt 和 `modeling_guide.md §Figure Style` 的任务层，而不改颜色/字体硬规范。

建议新增的 Step 8 选图规则：

1. 每个子问题至少有一个最终交付视觉锚点：主结果图或主结果表。
2. 几何/物理判据必须有解释图；复杂公式前先用图解释对象关系。
3. 优化/搜索结论必须有可信度图：目标函数、可行边界、收敛曲线、算法对比表四选一。
4. 空间分布结果优先用统一坐标和统一色标的多宫格散点图。
5. 图服务最终采信口径；粗网格、失败分支、退化解只可作为验证或局限性图，不能抢主图位置。
6. `visualization_log.md` 中每张图都要写明“叙事作用”：解释模型、报告结果、验证可信度、展示局限四选一。

建议不要接入的内容：

1. 不要求每篇论文必须使用现实照片。2025A 的现实烟幕照片只用于背景导入，不是核心建模图。
2. 不强制图越多越好。优秀论文的共同点是图有任务，而不是图数量多。
3. 不复制优秀论文的颜色。系统已有 `modeling_guide.md §Figure Style`，应保持统一论文级视觉规范。
