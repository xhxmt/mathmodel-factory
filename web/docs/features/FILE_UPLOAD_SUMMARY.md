
#### 新增端点
```python
POST /api/upload/problem
```

**功能**：
- 接收文件上传（multipart/form-data）
- 验证文件格式（.pdf, .md, .PDF, .MD）
- 验证文件大小（≤100MB）
- 生成唯一文件名（时间戳前缀）
- 保存到 `uploads/` 目录
- 返回服务器文件路径

**返回示例**：

## 🚀 使用流程

### 上传文件方式（推荐）

1. 点击 **"➕ 新建项目"**
2. 输入项目名称（如 `cumcm2024_a`）
3. 选择 **"📤 上传文件"** 标签
4. **上传文件**：
   - 点击上传区域，选择文件，或
   - 直接拖拽文件到上传区域
5. 等待上传完成（显示进度条）
6. 勾选选项（可选启用人工咨询）
7. 点击 **"创建项目"**

### 指定路径方式（传统）

1. 点击 **"➕ 新建项目"**
2. 输入项目名称
3. 选择 **"📁 指定路径"** 标签
4. 输入服务器文件路径
5. 点击 **"创建项目"**

## 🔒 安全措施

1. ✅ **文件类型白名单**：仅允许 `.pdf`, `.md`, `.PDF`, `.MD`
2. ✅ **文件大小限制**：最大 100 MB
3. ✅ **身份认证**：需要登录才能上传
4. ✅ **文件名清理**：添加时间戳前缀避免冲突
5. ✅ **目录隔离**：上传文件存储在独立的 `uploads/` 目录
